1. Visão Geral da Certificação Solutions Partner

A designação Solutions Partner with certified software é o mais alto nível de reconhecimento da Microsoft para ISVs (Independent Software Vendors). Ela substituiu o antigo sistema Gold/Silver e exige que o parceiro demonstre capacidade em três categorias: Performance, Skilling e Customer Success [citation:2][citation:3].

🎯 Objetivo: Transformar o EcoSystem Microsoft (27 agentes) em uma solução certificada pela Microsoft, desbloqueando benefícios exclusivos de Go-to-Market, co-sell e incentivos financeiros.

📊 Áreas de Designação

As seis áreas de designação do programa Microsoft Solutions Partner são [citation:2][citation:4]:

  • Data & AI (Azure) — soluções de dados e IA no Azure
  • Infrastructure (Azure) — infraestrutura em nuvem
  • Digital & App Innovation (Azure) — modernização de aplicações
  • Business Applications — Dynamics 365 e Power Platform
  • Modern Work — Microsoft 365 e produtividade
  • Security — soluções de segurança Microsoft

Nosso foco: Data & AI (Azure)

🏆 Por que Buscar a Certificação?

  • Credibilidade: 68% dos clientes avaliam certificações ao buscar soluções de software [citation:5]
  • Visibilidade: Badge digital aparece ao lado da sua solução no Microsoft Marketplace [citation:5]
  • Co-sell: Vendedores da Microsoft têm incentivos para vender sua solução [citation:7]
  • Incentivos: Acesso a até US$ 150.000 em créditos Azure e benefícios ISV Success [citation:9][citation:11]

2. Requisitos e Pontuação para Certificação

Para obter a designação, a empresa precisa atingir 70 pontos no Partner Capability Score, com pelo menos 1 ponto em cada uma das 5 métricas [citation:1][citation:3].

Categoria Métrica Pontuação Máxima Descrição
Performance Net Customer Adds 30 pontos Cada novo cliente que gera consumo Azure ≥ US$ 1.000 (Enterprise) ou ≥ US$ 500 (SMB) [citation:1]
Skilling Certificações Intermediárias ~20 pontos Número de colaboradores com certificações Associate (ex: Azure Administrator Associate) [citation:2]
Skilling Certificações Avançadas ~20 pontos Número de colaboradores com certificações Expert (ex: Azure Solutions Architect Expert) [citation:2]
Customer Success Usage Growth ~15 pontos Crescimento ano a ano no consumo Azure dos clientes [citation:2]
Customer Success Deployments ~15 pontos Número de clientes com implantações de workloads Azure [citation:2]

🔑 Pontos Críticos:

  • É necessário ter pelo menos 1 ponto em cada métrica (não pode zerar nenhuma) [citation:1][citation:3]
  • Classificação Enterprise: para parceiros com ACR ≥ US$ 1 milhão nos últimos 12 meses [citation:1]
  • Classificação SMB: para parceiros com ACR < US$ 1 milhão [citation:1]
  • O EcoSystem Microsoft atualmente se enquadra como SMB, com metas mais acessíveis [citation:1][citation:2]

3. Etapas para Certificação

3.1 ISV Success — O Ponto de Partida

O ISV Success é um programa de 12 meses da Microsoft para ISVs que desejam construir, publicar e vender aplicações no Marketplace. Ele é o primeiro passo para a certificação Solutions Partner [citation:9][citation:12].

Benefícios do ISV Success:

  • Pacote de benefícios no valor de US$ 126.000 — gratuito no primeiro ano [citation:9][citation:11]
  • Créditos Azure, ferramentas de desenvolvimento, consultoria 1:1 [citation:9]
  • Acesso antecipado a ferramentas de IA da Microsoft [citation:9]

Critérios de Elegibilidade: A empresa deve ter um projeto de desenvolvimento B2B em andamento, com compromisso de publicar no Marketplace em até 12 meses [citation:12].

3.2 Construir e Publicar no Marketplace

📦 Publicação de Ofertas

  • Publicar ofertas transacionáveis no Microsoft Marketplace [citation:5]
  • As ofertas devem ser soluções de software (não serviços profissionais) — serviços profissionais não contam para as métricas de certificação [citation:5]
  • O EcoSystem Microsoft já está em processo de publicação das ofertas SaaS

📈 Geração de Receita

  • A receita gerada no Marketplace é a métrica central para o Partner Capability Score
  • Net Customer Adds: cada novo cliente que gera consumo Azure ≥ US$ 500 (SMB) [citation:1]
  • O Marketplace gera um multiplicador de US$ 6,26 em valor extra para cada US$ 1 transacionado

3.3 Obter Certificações da Equipe

Nível Certificação Recomendada Exame Para o EcoSystem
Intermediária Azure Administrator Associate AZ-104 ✅ Fundamental para Data & AI [citation:2]
Intermediária Azure Developer Associate AZ-204 ✅ Para desenvolvimento dos agentes
Intermediária Azure AI Engineer Associate AI-102 ✅ Essencial para agentes de IA
Avançada Azure Solutions Architect Expert AZ-305 ✅ Necessário para pontuação máxima [citation:2]
Avançada Azure DevOps Expert AZ-400 ✅ Para integração contínua

3.4 Demonstrar Customer Success

Métricas de Customer Success exigidas [citation:1][citation:2]:

  • Usage Growth: Crescimento ano a ano no consumo Azure dos clientes
  • Deployments: Número de clientes com implantações de workloads Azure
  • Case Studies: Histórias de sucesso documentadas

Como o EcoSystem vai atender:

  • Os agentes geram consumo Azure (APIs, serviços de IA)
  • Clientes implantam os agentes em suas assinaturas Azure
  • Documentar cases com clientes iniciais

4. Benefícios da Certificação Solutions Partner

🏷️ Badging e Visibilidade

  • Badge digital aparece ao lado da solução no Marketplace [citation:5]
  • Certified letter e award logo disponíveis no Logo Builder [citation:5]
  • 68% dos clientes avaliam certificações ao buscar software [citation:5]

🤝 Co-sell e Go-to-Market

  • Solution Play Cards para vendedores da Microsoft [citation:5]
  • Vídeo animado de 30 segundos para marketing [citation:5]
  • Acesso a Marketing Concierge para suporte personalizado [citation:5]
  • Vendedores da Microsoft têm incentivos (PRACR) para vender sua solução [citation:7]

💰 Incentivos Financeiros

  • Azure Migrate & Modernize — até US$ 150.000 em incentivos [citation:7][citation:9]
  • Azure Innovate — incentivos para novas aplicações com IA [citation:7]
  • ISV Success Premium — até US$ 150.000 em cash rewards para soluções certificadas [citation:9]

🚀 Expansão de Canais

  • Resale-ready: habilitar parceiros para revender seus agentes [citation:5]
  • Acesso ao Partner Reported ACR (PRACR) — joint-sales com Microsoft [citation:7]
  • Enhanced seller discovery: solução aparece como "certified software" para vendedores Microsoft [citation:7]

5. Checklist Completo para Certificação

✅ Pré-requisitos ISV Success

  • Inscrever-se no ISV Success (gratuito no primeiro ano) [citation:11]
  • Confirmar elegibilidade: B2B, Microsoft Cloud, publicar no Marketplace [citation:12]
  • Designar Engagement Manager [citation:9]

✅ Publicação no Marketplace

  • Publicar ofertas transacionáveis (SaaS/Container) [citation:5]
  • Garantir que ofertas sejam soluções (não serviços profissionais) [citation:5]
  • Configurar dimensões de cobrança para metered billing

✅ Skilling — Certificações

  • Equipe com certificação AZ-104 (Azure Admin) [citation:2]
  • Equipe com certificação AI-102 (Azure AI Engineer)
  • Equipe com certificação AZ-305 (Solutions Architect) [citation:2]
  • Manter certificações ativas (renovar anualmente)

✅ Performance — Clientes

  • Fechar contratos com clientes que gerem consumo Azure
  • Documentar net customer adds (crescimento mensal)
  • Acompanhar ACR (Azure Consumed Revenue) dos clientes [citation:1]

✅ Customer Success

  • Documentar cases de sucesso com clientes
  • Demonstrar usage growth (crescimento de consumo) [citation:2]
  • Demonstrar deployments (implantações bem-sucedidas)

✅ Go-to-Market

  • Ativar Solution Play Cards no Partner Center [citation:5]
  • Baixar badge e materiais de marketing (Logo Builder) [citation:5]
  • Engajar com Marketing Concierge para suporte personalizado [citation:5]
