# Implementação: Máquina de Vendas com LinkedIn → Site → AION

## Contexto
Temos o Ecosystem AION (71 agentes) com integração LinkedIn via MCP. O site de vendas (landing page) está pronto para receber tráfego. Agora precisamos estruturar o fluxo completo de prospecção, qualificação e conversão, onde os agentes do AION atuam como uma máquina de vendas autônoma que:

1. Prospecta leads no LinkedIn
2. Qualifica leads e enriquece dados
3. Conduz leads para o site via conteúdo personalizado
4. Captura leads no site e inicia nutrição
5. Qualifica para demonstração e venda
6. Ativa agentes do AION para o cliente

# webhooks/site_webhook.py
from fastapi import FastAPI, Request
from src.database.supabase_client import supabase
from src.agents.sales.qualify import LeadQualifier

app = FastAPI()

@app.post("/webhook/lead")
async def receive_lead(request: Request):
    """
    Recebe leads do site e inicia processo de qualificação
    """
    data = await request.json()
    
    # 1. Salva lead no Supabase
    lead = supabase.table("leads").insert({
        "name": data["name"],
        "email": data["email"],
        "company": data["company"],
        "phone": data["phone"],
        "source": "site_landing_page",
        "status": "new",
        "created_at": datetime.now().isoformat()
    }).execute()
    
    # 2. Dispara qualificação
    qualifier = LeadQualifier()
    score = await qualifier.score_lead(lead.data[0])
    
    # 3. Se alto score, agenda demonstração
    if score >= 80:
        await schedule_demo(lead.data[0])
    
    # 4. Ativa nutrição automática
    await start_nurture_campaign(lead.data[0])
    
    return {"status": "success", "lead_id": lead.data[0]["id"]}
