## 1. VISÃO GERAL DO PROJETO

### 🎯 Objetivo
Desenvolver um sistema multiagente de IA para o setor de Arquitetura, Engenharia e Construção (AEC), com venda por assinatura e cross-selling automático entre os agentes.

### 📊 Nicho de Mercado
**Setor AEC (Arquitetura, Engenharia e Construção)**
- Apenas 4% das empresas usam IA em escala
- Demanda crescente por automação
- Processos manuais e fragmentados

### 🧠 Tecnologia Base
- **LLM Principal:** DeepSeek-V4-Flash (via API OpenAI compatible)
- **Framework:** Python com arquitetura multiagente
- **Orquestrador:** Sistema central que gerencia todos os agentes
