1. Opik — Observabilidade e Avaliação Open-Source

O Opik (built by Comet) é a ferramenta mais completa para monitoramento de agentes, com integração nativa para frameworks como LangChain, CrewAI, Autogen e DeepSeek .

Instalação:
bash

pip install opik

Integração com AION:
python

from opik import Opik
from opik.evaluation.metrics import Hallucination, AnswerRelevance, ContextPrecision
import opik

# Configuração
opik.configure(api_key="YOUR_API_KEY")

# Logging de traces
@opik.trace
async def execute_agent(agent_id: str, input_data: dict):
    # Seu código existente
    result = await agent.execute(input_data)
    return result

# Avaliação com LLM-as-a-Judge
metric = Hallucination()
score = metric.score(
    input="Qual a vida útil de um motor elétrico?",
    output="A vida útil média é de 15 anos",
    context=["Motores elétricos duram entre 10 e 20 anos"]
)

2. Agent Health Monitoring (Salesforce)

Baseado na solução da Salesforce para monitoramento de agentes :

Métricas Essenciais:

    Agent Error Rate — % de respostas que falham

    Average Interaction Latency — tempo de resposta

    Escalation Rate — % de sessões que escalam para humano

Implementação:
python

class AgentHealthMonitor:
    def __init__(self):
        self.metrics = {
            "error_rate": 0,
            "latency": 0,
            "escalation_rate": 0
        }
        self.thresholds = {
            "error_rate": 0.05,  # 5% max
            "latency": 2.0,      # 2 segundos max
            "escalation_rate": 0.15  # 15% max
        }
    
    def check_health(self, agent_id: str, session_data: dict):
        # Calcula métricas da sessão
        error_rate = self._calculate_error_rate(session_data)
        latency = self._calculate_latency(session_data)
        escalation_rate = self._calculate_escalation_rate(session_data)
        
        # Alerta se threshold ultrapassado
        if error_rate > self.thresholds["error_rate"]:
            self._send_alert(agent_id, "error_rate", error_rate)
        
        return {
            "agent_id": agent_id,
            "status": "healthy" if all([
                error_rate <= self.thresholds["error_rate"],
                latency <= self.thresholds["latency"],
                escalation_rate <= self.thresholds["escalation_rate"]
            ]) else "unhealthy",
            "metrics": {
                "error_rate": error_rate,
                "latency": latency,
                "escalation_rate": escalation_rate
            }
        }

3. Azure Monitor + Application Insights (Microsoft)

Para integração com o ecossistema Microsoft :

Métricas Monitoradas:

    Run rates e run duration

    Task failures e error rates

    Token usage e custos

    CPU, memory, network usage

Configuração:
python

from azure.monitor import ApplicationInsightsClient

class AzureMonitor:
    def __init__(self, connection_string: str):
        self.client = ApplicationInsightsClient(connection_string)
    
    def track_agent_execution(self, agent_id: str, duration: float, success: bool):
        self.client.track_event(
            name="AgentExecution",
            properties={
                "agent_id": agent_id,
                "duration": duration,
                "success": success
            }
        )
    
    def track_token_usage(self, agent_id: str, tokens: int, model: str):
        self.client.track_metric(
            name="TokenUsage",
            value=tokens,
            properties={"agent_id": agent_id, "model": model}
        )

4. Oracle Fusion AI Agent Studio — Monitoramento e Avaliação

A Oracle oferece um framework completo com métricas específicas :

Métricas de Monitoramento:

    Accuracy (precisão das respostas)

    Latency (P50 e P99)

    Token Usage (total, input, output)

Avaliação Estruturada:

    Evaluation Sets — testes com perguntas e respostas esperadas

    Run Comparison — comparar versões de agentes

    RAG Metrics — Groundedness, Answer Relevance, Context Relevance

Implementação:
python

class OracleEvaluation:
    def __init__(self):
        self.evaluation_sets = {}
    
    def create_evaluation_set(self, agent_id: str, test_cases: list):
        self.evaluation_sets[agent_id] = test_cases
    
    def run_evaluation(self, agent_id: str):
        test_cases = self.evaluation_sets.get(agent_id)
        results = []
        for test in test_cases:
            response = agent.execute(test["input"])
            score = self._calculate_accuracy(response, test["expected"])
            results.append({"test": test, "response": response, "score": score})
        
        return {
            "agent_id": agent_id,
            "accuracy": sum([r["score"] for r in results]) / len(results),
            "detailed_results": results
        }

📋 RELATÓRIO COMPLETO DOS AGENTES
Estrutura do Relatório
python

class AgentReport:
    def __init__(self):
        self.report = {
            "timestamp": datetime.now().isoformat(),
            "agents": {},
            "summary": {}
        }
    
    def generate_report(self):
        for agent_id in self.get_all_agents():
            report_data = self._collect_agent_data(agent_id)
            self.report["agents"][agent_id] = report_data
        
        self.report["summary"] = self._generate_summary()
        return self.report
    
    def _collect_agent_data(self, agent_id: str) -> dict:
        return {
            "id": agent_id,
            "name": self._get_agent_name(agent_id),
            "group": self._get_agent_group(agent_id),
            "metrics": self._get_metrics(agent_id),
            "evaluation": self._run_evaluation(agent_id),
            "health": self._check_health(agent_id),
            "performance": self._get_performance(agent_id),
            "cost": self._get_cost(agent_id),
            "recommendations": self._generate_recommendations(agent_id)
        }
