EcoSystem AEC + Regulatory

Empresa: Global Match Engenharia de Produção

Versão: 3.0.0  |  Azure Marketplace SaaS

1. Descrição do Produto

Plataforma SaaS composta por mais de 50 agentes de IA generativa especializados em:

  • Obrigações regulatórias brasileiras — NR-1 Psicossocial, LGPD, Igualdade Salarial, Canal de Denúncias, CBS/IBS, Anticorrupção
  • Relatórios ESG — IFRS S1/S2, Inventário de Carbono, Escopo 3, CBAM
  • Processos de engenharia e construção (AEC) — Spec Analyst, Procurement, Inventory, BIM Coordination
  • Integração com ERPs — Microsoft Dynamics 365, Salesforce Agentforce, Oracle Fusion, SAP S/4HANA

Os agentes operam de forma hierárquica e colaborativa utilizando os protocolos A2A (Agent-to-Agent) da Google e MCP (Model Context Protocol) da Anthropic, permitindo orquestração inteligente de tarefas complexas.

3. Fluxo de Funcionamento

  1. Acesso: Cliente acessa o dashboard web (Vercel) e seleciona um agente ou descreve um objetivo em linguagem natural
  2. Requisição: A requisição é enviada via HTTPS para a API FastAPI hospedada no Render.com
  3. Orquestração: O Master Orchestrator analisa o objetivo, identifica as obrigações legais aplicáveis e delega para o(s) agente(s) especializado(s)
  4. Execução: Cada agente consulta o LLM apropriado (DeepSeek, Gemini ou Claude) com contexto recuperado do banco vetorial (RAG)
  5. Revisão: O Quality Critic valida o output contra a norma vigente antes da entrega ao cliente
  6. Entrega: Resultado é armazenado, apresentado no dashboard e notificado por e-mail

Nenhum componente é implantado no ambiente Azure do cliente. Toda a plataforma é SaaS multi-tenant, acessada via navegador.
